Crystal Collector

Crystal Collector is a game where the player must use the crystal buttons to add up to the total number displayed by the computer. The numbers behind the crystals are randomly generated each new game and the player must be careful to add correctly or they lose the game. The overall game is designed with HTML5, CSS3, Javascript and jQuery. Click on the button below to see the game in action!

Guess The Letter

Guess the letter is a project where the user is tasked to guess what letter the computer is thinking of. The user gets 9 tries to see if they can correctly guess the letter the computer is thinking of. The guesses can be between the letters A - Z. If they guess correctly within the 9 attempts they win! Else, they lose but you can automatically start guessing again. This was built with Javascript, CSS3 and HTML5. To see the project and attempt to guess the letter, click on the button below.
